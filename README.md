# MVP Landing

랜딩 페이지와 MVP 검증을 위한 매우 작은 Next.js 프로젝트입니다.

Vercel 배포 확인과 간단한 유입 로그 실험을 빠르게 진행하기 위한 용도입니다.

## Local Development

의존성을 설치하고 개발 서버를 실행합니다.

```bash
npm install
npm run dev
```

로컬 환경변수는 `.env.local`에만 두고 Git에는 올리지 않습니다.

## Available Commands

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Collaboration

- `main`: 운영 브랜치
- `feature/*`: 작업 브랜치
- PR을 올린 뒤 Vercel Preview에서 확인하고 merge합니다.
